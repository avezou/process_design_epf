Main Description

Rule design address the discipline of designing the rule set structure, which includes the parameters input and output to the rule set, but also the logic of execution of the rules using rule flow or other mechanism. The complexity of a rule set implementation is also coming from the data modeling, therefore the design discipline includes a set of activity to design and build the data model or fact model.

The task of defining where to implement and enforce the business rule is also part of the design discipline.
