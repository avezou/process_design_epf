Disciplines: Rule Authoring
Relationships
RolesPrimary Performer:

Additional Performers:
Main Description
  • Formally log any issues and decisions done during the rule analysis and the rule authoring activities.
  • The issue log is used to manage feedbacks, and to improve the rule set quality.
  • As a social process, people will change their mind, it is a good practice to log decision on a timely manner.
